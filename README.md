## ZooTotemBot - Telegram-бот Московского зоопарка

Программа представляет собой интерактивного Telegram-бота для зоопарка.
Бот позволяет пользователям проходить викторину, определять свое тотемное животное и связываться с сотрудниками

## Возможности программы

- Интерактивная викторина с вопросами.
- Определение тотемного животного по ответам пользователя.
- Предоставление фотографий и интересных фактов о животных.
- Информирование о программе опеки над животными в зоопарке.
- Возможность повторного прохождения викторины.

## Новый функционал

- Встроенный контактный механизм для прямой связи с сотрудниками зоопарка.
- Централизованная система сбора отзывов о работе бота.
- Автоматическая пересылка результатов викторины сотруднику при запросе контакта.

## Установка и запуск программы
- Клонируйте репозиторий на свой компьютер:


**bash
git clone https://github.com/ваш_логин/ZooTotemBot.git
cd ZooTotemBot**

- Установите необходимые библиотеки и зависимости:
  **bash
  pip install python-telegram-bot python-dotenv**

- Настройте конфигурацию бота:
Создайте текстовый файл с именем .env в корневой папке проекта
Добавьте в созданный файл ваш уникальный токен Telegram:

**TELEGRAM_TOKEN=ваш_токен_бота**

- Запустите основной скрипт бота:
**python bot.py**
